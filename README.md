# Chat CTA for ConcreteCMS

Chat CTA is a lightweight ConcreteCMS package that adds a floating chat/contact button to all public-facing pages.

## Features

- Floating global CTA button
- Random number selection
- Weighted distribution between operators
- Per-number click counter
- WhatsApp support (more providers planned)
- Uses built-in Font Awesome
- No manual block placement required
- Clean ConcreteCMS 9 dashboard integration

## Current Provider Support

- WhatsApp

## Planned Providers

- Telegram
- Signal
- SMS
- Phone
- Email
- Custom URL

## Installation
1. Download & unpack the ZIP File

2. Move the package into:
   /packages

3. Rename the folder to /chat_cta
   
4. Install the package from:

   Dashboard → Extend ConcreteCMS

## Dashboard

Settings page:

Dashboard → System & Settings → Chat CTA
